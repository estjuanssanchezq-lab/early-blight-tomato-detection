# early-blight-tomato-detection
Detección de enfermedades en hojas de tomate mediante procesamiento digital de imágenes. El proyecto analiza síntomas del tizón temprano (Early Blight) usando técnicas como filtrado, segmentación y análisis de color, con el objetivo de lograr una identificación temprana y objetiva que apoye la toma de decisiones en el cultivo.

# Información

Dataset utilizado:
https://data.mendeley.com/datasets/tywbtsjrjv/1

Clases utilizadas:
- Tomato Early Blight
- Tomato Healthy
